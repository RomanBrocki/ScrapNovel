Script cominterface TK de srap de webnovel do site https://novelbin.com/ por meio do selenium.
A partir do site do capítulo inicial, e do site do capítulo final, gera um docx com o conteúdo.
Medidas antiscrapping geram texto "marca dágua" no meio do conteúdo. 
Com base de dados das marcas dágua encontradas, o script possui recurso para retirar estas do conteúdo, 
mas algumas novas podem ocorrer.

TK gerado por meio do figma com interpretação pelo https://github.com/Proxlight/Proxlight-Designer-v2.
A versão atual do mesmo não suporta mais esse tipo de interação.

Indicado ter o chrome instalado, com perfil designado com extensão de adblocker.
Verificar o arquivo ClasseEbook para mais informações.

Instruções para instalação de bibliotecas com titulos diferentes no import x install:
pip install webdriver-manager
pip install python-docx
